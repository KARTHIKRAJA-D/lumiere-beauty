LUMIÈRE BEAUTY — Images Folder
================================
This folder is for your custom images.

Recommended images to add:
- hero-portrait.jpg      (Main hero image, portrait orientation, ~800x1000px)
- artist-photo.jpg       (Sofia's professional photo, ~600x800px)
- gallery-1.jpg          (Portfolio shots, ~600x800px each)
- gallery-2.jpg
- gallery-3.jpg
- logo.svg               (Optional SVG logo)

Image tips for best results:
- Use JPG/WebP for photos (compress to <200KB each)
- Use PNG for logos with transparency
- Recommended format: WebP for web performance
- Free stock photos: unsplash.com, pexels.com

The website currently uses Unsplash CDN images for demo.
Replace the <img src="https://..."> URLs in index.html
with your own images: <img src="images/your-photo.jpg">
